## 0.6官方的策略包导入

上面配置过程较为繁琐，jrasp 官方提供了可以导入的完整包。
下载通用策略包：
```java
wget https://www.download.jrasp.com/v1.1.3/export_%E9%BB%98%E8%AE%A4%E9%85%8D%E7%BD%AE.zip
```
